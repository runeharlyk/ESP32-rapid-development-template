## Getting started

1. Fork the project

1. Clone and open the new project

   ```sh
   git clone https://github.com/{USERNAME}/ESP32-rapid-development-template
   ```

1. Install dependencies

   ```sh
   cd app
   npm install
   ```

## Usage

1. Run the app

   ```sh
   npm run dev
   ```
